# CurrencyCalculator
Currency Calculator REST API
application for calculating currency exchange
GET request example:
http://localhost:8080/currency/calculate?amount=100&actualCurrency=EUR&targetCurrency=USD

Possible currencies codes:
    EUR,
    USD,
    GBP,
    AUD,
    CAD,
    HUF,
    CHF,
    JPY,
    SEK,
    CZK,
    DKK,
    NOK,
    XDR;
